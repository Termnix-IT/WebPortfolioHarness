# Public Repository Policy

このリポジトリは公開可能な情報だけを置く前提で運用します。

## Allowed

- 公開済みのGitHubリポジトリURL
- 公開済みのQiita記事URL
- 公開ポートフォリオURL
- 抽象化された職務経歴
- 公開可能な技術スキル
- 取得済み資格
- 個人検証プロジェクトの概要

## Not Allowed

- 住所
- 電話番号
- 非公開メールアドレス
- 顧客名
- 案件名
- 契約情報
- 内部ネットワーク構成
- 非公開IPアドレス
- 認証情報
- API keys
- tokens
- 障害の詳細な内部情報
- 勤務先固有の運用手順

## Redaction Guidelines

- 固有名詞は業種や役割に置き換える
- 内部構成は「監視環境」「ネットワーク機器」「業務システム」のように抽象化する
- 数値実績は公開してよい範囲だけにする
- 公開可否に迷う内容は書かない

## AI Usage Notes

AIに渡す前に、入力内容に非公開情報が含まれていないか確認してください。

AIが生成した文章は、事実確認と公開可否確認を行ってから採用してください。
